# Производство · прототипы интерфейсов

Пять самодостаточных HTML-экранов модуля «Производство» (сборка АКБ под заказ).
Открываются двойным кликом, без зависимостей и сборки. Все данные — демо, живут в сессии браузера.

| Файл | Экран |
|---|---|
| `proizvodstvo_voronka.html` | Воронка сборки: 6 этапов MTO, drag-and-drop, нормо-часы на карточках, ворота комплектации, карта изделия (маршрут · BOM · нормо-часы · ОТК) |
| `proizvodstvo_vyrabotka.html` | Выработка и оценка: табель нормо-часов, расчёт ЗП (оклад × дни/22 + выработка × 6,25 BYN), лидеры по вкладу |
| `proizvodstvo_planirovanie.html` | Планирование: годовая матрица SKU × 12 мес (штуки/н.ч), загрузка против мощности 704 н.ч, правка плана по ячейкам, добавление/редактирование/удаление позиций |
| `proizvodstvo_normy.html` | Нормы и нормативы: справочник 35 изделий + 24 операций, статусы утверждения, состав работ изделия с правкой и «Σ как норма» |
| `proizvodstvo_otk.html` | ОТК: очередь (100% / AQL / повторный), паспорт изделия, чек-лист с допусками, решения принять/доработка/брак, эскалация закупщику (претензия поставщику) и руководителю |

**Принципы прототипа:** без ИИ (блоки «Помощники ИИ · Итерация 1» — пунктирные плейсхолдеры), валюта BYN, нормо-час — базовая единица учёта, единый дизайн-шелл ERP (Manrope, 236/1fr/312).

**Проверка:** 99 функциональных тестов (Playwright/Chromium) в `tests/` корня репозитория — клики, правки, drag-and-drop, модальные окна, пересчёты, решения ОТК. Запуск: `npm install && npx playwright install chromium && npm test`.
